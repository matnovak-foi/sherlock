Load Marking

NOTE: Loading and saving marking is not yet supported in the natural language detection modes.

To load markings from a file.

In the Display Matches Window,
- Click Load Marking on the File menu, select the file which stores your marking and confirm the selection. The marking should be loaded and is reflected in both Match Table & Match Graph views. Note: If you load a wrong marking file, Sherlock will not detect this, however the system will not crash in this case, but the markings won't be the one you are expecting.
